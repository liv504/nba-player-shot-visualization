# 🏀 NBA Shot Chart Visualizer

Interactive NBA shot chart built with **Flask + D3.js**.
Visualize shot locations for any player across seasons with filters for playoffs, opponents, and more.

---

## 🚀 Features

* 🔍 Search by player name (auto-resolves player ID)
* 📅 Filter by season range
* 🏀 Regular season or playoffs
* 🆚 Filter by opponent teams
* 🎯 Animated shot plotting
* 🟢 Green = made shots, 🔴 Red = missed shots

* **Frontend:** D3.js, HTML/CSS
* **Data:** NBA Stats API

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/nba-shot-chart.git
cd nba-shot-chart
pip install -r requirements.txt
python app.py
```

Then open:

```
http://127.0.0.1:5000
```
